# ghostship-qbittorrent

CLI utility for Qbittorrent API.

## Environment Variables

- `QBITTORRENT_URL`: The base URL of your qBittorrent instance (e.g., `http://localhost:8080`). Required.
- `QBITTORRENT_USER`: Your qBittorrent username. Optional if authentication is disabled.
- `QBITTORRENT_PASS`: Your qBittorrent password. Optional if authentication is disabled.
