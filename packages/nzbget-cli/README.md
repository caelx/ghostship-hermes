# ghostship-nzbget

CLI utility for Nzbget API.

## Environment Variables

- `NZBGET_URL`: The base URL of your NZBGet instance (e.g., `http://localhost:6789`). Required.
- `NZBGET_USER`: Your NZBGet username. Optional if authentication is disabled.
- `NZBGET_PASS`: Your NZBGet password. Optional if authentication is disabled.
